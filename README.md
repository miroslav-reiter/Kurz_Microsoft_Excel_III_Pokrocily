## Obsah
### Microsoft Excel má 3 typy Hárkov (Sheets)
1. **Pracovný Hárok (Worksheet) (Shift + F11)**
2. Makro Hárok (Macro Sheet) (Ctrl + F11)
3. Grafový Hárok (Graph Sheet) (F11)
